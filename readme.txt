Кандидатам на вакансию разработчика Symfony предлагается задача – собрать на этом фреймворке простой API.

Исходный код должен быть залит в публичный bitbucket или github-репозиторий.

Необходимо реализовать API для выдачи новостей в формате JSON с возможностью фильтрации по году+месяцу (если указывается одно из полей, то второе становится обязательным) и тегам.

Поля новости: название, текст, фото, дата публикации, теги. Примеры тегов: #экономика, #политика и т.д. Метод должен возвращать все поля новости и включённых в неё тегов

Сортировка по дате публикации (новые раньше). Разбиение на страницы по 5 новостей.

Для начального заполнения нужна миграция, которая добавляет 7 новостей с 2-3 тегами. Админка будет плюсом, но делать её необязательно.

Будет оцениваться стиль кода, простота решений, история git репозитория, документация по развертыванию.
